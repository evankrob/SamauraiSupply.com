# SamuraiSupply.com

The archived catalog of the original Samurai Supply sword shop (est. 2009), rebuilt as a
static Amazon-affiliate reference site. Hosted on GitHub Pages at
[samuraisupply.com](https://samuraisupply.com).

## How it works

- **519 product pages** regenerated from the original store scrape, keeping the same
  file names so old URLs and inbound links still resolve.
- Every product page links to a live Amazon search for the item with the affiliate tag
  `ralefive-20` (`rel="sponsored nofollow"`).
- Zero JavaScript, one 7 KB stylesheet (`assets/site.css`), system fonts only.
- Images that were never captured in the scrape render as a styled 刀 placeholder
  instead of broken images.
- Dead ecommerce machinery (cart, checkout, admin, jQuery, PHPSESSID artifacts) removed;
  obsolete pages (store policies, international orders, etc.) are meta-refresh redirect stubs.

## Rebuilding

The site is generated by the scripts in `tools/`:

```sh
python3 tools/generate.py    # regenerate every page from tools/catalog.json
python3 tools/check_links.py # verify all internal hrefs/srcs resolve
```

(`tools/extract.py` was the one-time parser that built `catalog.json`/`pages.json` from the
original store scrape; it can't be re-run now that the old pages have been replaced.)

`tools/catalog.json` is the product database (name, SKU, brand, category, images,
description HTML). Edit `tools/generate.py` for template/CSS changes, then rerun it.
Requires `beautifulsoup4` and `Pillow`.
